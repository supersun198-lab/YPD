# 深圳市熠普达科技有限公司官网静态版

这是一个企业官网 / 产品展示页的纯前端静态版本，内容围绕“深圳市熠普达科技有限公司”和 LED 相关电子产品编写。

## 文件说明

- `index.html`：页面结构与文案内容
- `styles.css`：页面样式、响应式布局与动画
- `script.js`：移动端导航与滚动显示动画

## 如何预览

直接用浏览器打开：

```text
web/index.html
```

也可以在编辑器中安装 Live Server 一类插件进行本地预览。

## 后续可补充内容

当前以下内容为占位，拿到真实资料后可直接替换：

- 联系电话
- 电子邮箱
- 详细公司地址
- 产品真实名称、图片、型号、参数
- 公司资质、案例、客户行业
- 备案号
